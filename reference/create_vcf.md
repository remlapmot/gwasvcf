# Create GWAS vcf

Create GWAS vcf

## Usage

``` r
create_vcf(
  chrom,
  pos,
  nea,
  ea,
  snp = NULL,
  ea_af = NULL,
  effect = NULL,
  se = NULL,
  pval = NULL,
  n = NULL,
  ncase = NULL,
  name = NULL
)
```

## Arguments

- chrom:

  chrom vector

- pos:

  pos vector

- nea:

  nea vector

- ea:

  ea vector

- snp:

  Optional vector

- ea_af:

  Optional vector

- effect:

  Optional vector

- se:

  Optional vector

- pval:

  Optional vector

- n:

  Optional vector

- ncase:

  Optional vector

- name:

  Optional vector

## Value

vcf object
